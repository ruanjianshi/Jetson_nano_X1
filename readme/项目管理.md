## 4. 模块说明

### 4.1 main.py
主程序入口，负责初始化和调度各模块

### 4.2 config/config.yaml
配置文件，包含：
- 摄像头参数（分辨率、帧率、设备ID）
- 模型路径（ONNX、TensorRT模型）
- 推理参数（批处理大小、置信度阈值）
- 日志配置（级别、输出路径）

### 4.3 utils/
- **camera.py**: CSI/USB摄像头接口封装，支持GStreamer
- **logger.py**: 多级别日志记录（DEBUG, INFO, WARNING, ERROR）
- **hardware.py**: Jetson硬件接口（GPIO、I2C、串口、风扇控制）

### 4.4 inference/
AI推理模块，支持：
- TensorRT引擎加载
- 模型预处理和后处理
- GPU加速推理
- 结果可视化

## 8. 常见问题

### 8.1 GPU不可用
- 检查电源模式：`sudo nvpmodel -q`
- 启用MAXN模式：`sudo nvpmodel -m 0 && sudo jetson_clocks`

### 8.2 numpy崩溃
- 使用系统默认版本（1.17.4）
- 不要升级到1.19.x+

### 8.3 TensorFlow GPU问题
- TensorFlow 2.4.1 GPU检测有bug
- 建议使用PyTorch进行GPU加速
