## 技术实现

### 串口通信节点技术细节

**核心类：** `SerialCommNode`

**主要方法：**
- `detect_serial_port()` - 自动检测串口
- `connect_serial()` - 连接串口
- `disconnect_serial()` - 断开串口
- `reconnect_loop()` - 重连循环
- `tx_callback()` - 发送回调
- `rx_read()` - 读取数据
- `publish_status()` - 发布状态

**配置参数：**
```yaml
serial_port: auto        # 串口端口（auto为自动检测）
baud_rate: 115200        # 波特率
timeout: 1.0            # 超时时间
auto_reconnect: true     # 自动重连
reconnect_interval: 2.0  # 重连间隔
enable_echo: false      # 数据回显
```

## 遇到的问题和解决方案

### 问题1: 串口权限不足
**现象：**
```
[Errno 13] Permission denied: '/dev/ttyTHS1'
```

**解决方案：**
```bash
# 添加用户到dialout组
sudo usermod -a -G dialout jetson

# 登出后重新登录使权限生效
```

### 问题2: 文档冗余
**现象：**
- 多个GPIO相关文档内容重复
